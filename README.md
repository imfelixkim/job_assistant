# job-assistant

> 채용공고 분석 + 맞춤 이력서 최적화 워크플로우 — Claude Code 플러그인

채용공고를 입력하면, AI가 기업 리서치 → 공고 분석 → 매칭 분석 → 이력서 최적화 → 4관점 리뷰까지 11단계 대화형 워크플로우로 진행합니다.

## 설치

```bash
claude plugin add https://github.com/imfelixkim/job_assistant
```

또는 로컬에서 직접 사용:

```bash
git clone https://github.com/imfelixkim/job_assistant.git
claude --plugin-dir ./job_assistant
```

## 스킬 (3개)

### full-analysis — 전체 파이프라인 (핵심)

채용공고와 이력서를 받아 11단계 워크플로우를 대화형으로 진행합니다.

```
이력서 최적화해줘
[채용공고 URL 또는 텍스트]
```

**11단계 워크플로우:**
1. 채용공고 수집
2. 기업 리서치 (사업/재무/문화/포지션 맥락)
3. 채용공고 구조화 분석
4. 이력서 읽기
5. 매칭 분석 (강점/보완점/키워드)
6. 이직 동기 초안
7. 맞춤 이력서 최적화
8. 4관점 리뷰 (실무자/채용담당자/인사담당자/CEO)
9. 피드백 반영
10. 사용자 최종 승인
11. 파일 저장

### quick-review — 간편 리뷰

이력서를 4관점(실무자/채용담당자/인사담당자/CEO)에서 빠르게 리뷰합니다.

```
이력서 리뷰해줘
[이력서 텍스트 또는 파일 경로]
```

채용공고 없이도 일반적인 관점에서 리뷰 가능합니다.

### build-pdf — PDF 빌드 안내

마스킹된 이력서를 실제 개인정보로 치환하고 PDF로 변환하는 방법을 안내합니다.

```
PDF 만들어줘
[이력서 MD 파일 경로]
```

## 에이전트 (8개)

| 에이전트 | 역할 |
|----------|------|
| `job-analyzer` | 채용공고 텍스트를 구조화 분석 |
| `resume-matcher` | 채용공고와 이력서 매칭 분석 |
| `resume-optimizer` | 맞춤 이력서 최적화 작성 |
| `reviewer-practitioner` | 실무자(팀 동료) 관점 리뷰 |
| `reviewer-recruiter` | 채용담당자(리크루터) 관점 리뷰 |
| `reviewer-hr` | 인사담당자(HR) 관점 리뷰 |
| `reviewer-ceo` | CEO/경영자 관점 리뷰 |
| `company-researcher` | 기업 리서치 (사업/재무/문화) |

## 개인정보 보호

- 이력서 작성 시 개인정보는 마스킹 토큰(`[마스킹:성명]`, `[마스킹:연락처]` 등)으로 대체됩니다
- 마스킹된 상태로 모든 분석/리뷰가 진행됩니다
- PDF 변환 시에만 실제 정보로 치환됩니다 (로컬 실행)

## MCP 서버 연동 (프리미엄)

이 플러그인에는 `.mcp.json`이 포함되어 있어, MCP 서버와 함께 사용할 수 있습니다.

| | 플러그인만 (무료) | 플러그인 + MCP 서버 |
|---|---|---|
| 채용공고 분석 | Claude가 직접 분석 | Haiku 최적화 프롬프트 |
| 매칭 분석 | Claude가 직접 비교 | Sonnet 구조화 분석 |
| 4관점 리뷰 | 기본 관점 리뷰 | 상세 가중치 적용 리뷰 |
| 기업 리서치 | 기본 리서치 | 10개 소스 + 연봉 추정 |
| 비용 | $0 | 건당 ~$0.54 |

MCP 서버는 API 키 없이도 플러그인이 정상 동작합니다.
API 키(`ANTHROPIC_API_KEY`)를 설정하면 프리미엄 기능(상세 분석)이 활성화됩니다.

```bash
# MCP 서버 설치 (선택)
npm install -g job-service-mcp
```
